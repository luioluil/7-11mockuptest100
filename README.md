## ✨ Concept

> “เราต้องการให้ลูกค้าซื้อของได้เร็วที่สุด แต่ไม่รู้สึกเร่งรัด”

โปรเจกต์นี้จำลองประสบการณ์ซื้อสินค้าจาก 7-Eleven ในรูปแบบ e-commerce ที่  
**ค้นหาเร็ว (ฟิลเตอร์ไว)** • **เพิ่มตะกร้าได้ฉลาด** • **เช็คเอาต์ใน 3 สเต็ป**

โครงสร้างถูกออกแบบให้เป็น **Front-end Mockup พร้อมต่อ API**  
สามารถนำไปเชื่อมระบบจริง หรือ นำเสนอแนวคิดผลิตภัณฑ์ได้ทันที  

---

## ⚙️ Tech Stack

| Layer | Details |
|-------|----------|
| **Frontend** | HTML 5 / Bootstrap 5 / CSS Custom Variables |
| **Icons** | Bootstrap Icons 1.11 |
| **Font** | Poppins + Noto Sans Thai |
| **Animation** | Intersection Observer + CSS Transition |
| **Responsiveness** | Flex / Grid + Clamp Typography |
| **Version Control** | GitHub (Public Repository) |

---

## 🧩 Main Features

| หมวด | รายละเอียด |
|------|-------------|
| 🧠 Smart Filter | กรองหมวด โปร ช่วงราคา แท็ก + ค้นหาไทยแบบเรียลไทม์ |
| 🛒 Intelligent Cart | เพิ่มจำนวน เลือกออปชัน พร้อมแนะนำ similar / cross-sell |
| ⚡ Fast Checkout | เช็คเอาต์ 3 ขั้น + บันทึกข้อมูลใน LocalStorage |
| 📱 Mobile First | UI ออกแบบตามแนว Responsive จริงทุกขนาดจอ |
| 🧮 Practical Integration | สคีมาพร้อมต่อ API (สินค้า/ชำระเงิน/โลจิสติกส์) |
